# Расчет стоимости технического аудита

## Используемые API и сервисы

### 1. Google PageSpeed Insights API
- **Статус**: Бесплатный (с лимитами)
- **Лимиты**: 
  - 25,000 запросов в день (на один API ключ)
  - Нет платы за использование в пределах лимита
- **Использование в аудите**: 
  - 2 запроса на каждый URL (desktop + mobile)
  - Для клиники: 2 запроса (главная страница)
  - Для услуги: 2 запроса (страница услуги)

### 2. OpenAI API (GPT-4o-mini)
- **Модель**: gpt-4o-mini
- **Цены** (на декабрь 2025):
  - Input: $0.150 за 1M токенов (~$0.00015 за 1K токенов)
  - Output: $0.600 за 1M токенов (~$0.0006 за 1K токенов)
  - *Примечание: OpenAI сохранил стабильные цены на GPT-4o-mini в 2025 году*
- **Использование в аудите**:
  - Анализ llms.txt: ~1,500-2,500 токенов (input) + ~500-800 токенов (output)
  - Анализ технического аудита: ~2,000-3,500 токенов (input) + ~800-1,200 токенов (output)

### 3. HTTP запросы (бесплатно)
- robots.txt, sitemap.xml, llms.txt, HTML страницы
- Обычные HTTP запросы без платы

## Расчет для полного аудита клиники

### Компоненты аудита:
1. **Технический аудит главной страницы**:
   - PageSpeed Desktop: 1 запрос (бесплатно)
   - PageSpeed Mobile: 1 запрос (бесплатно)
   - HTML парсинг: 1 запрос (бесплатно)
   - Проверка файлов: robots.txt, sitemap.xml, llms.txt (бесплатно)
   - Проверка дубликатов: 3-4 HTTP запроса (бесплатно)

2. **AI анализ llms.txt** (если файл существует):
   - Input: ~2,000 токенов (средний размер llms.txt ~6,000 символов)
   - Output: ~600 токенов
   - **Стоимость**: (2,000 × $0.00015) + (600 × $0.0006) = $0.30 + $0.36 = **$0.66**

3. **AI анализ технического аудита**:
   - Input: ~2,500 токенов (форматированные данные аудита)
   - Output: ~1,000 токенов
   - **Стоимость**: (2,500 × $0.00015) + (1,000 × $0.0006) = $0.375 + $0.60 = **$0.975**

### Итого для полного аудита клиники:
- **PageSpeed API**: $0.00 (бесплатно)
- **OpenAI (llms.txt)**: $0.66
- **OpenAI (технический аудит)**: $0.975
- **HTTP запросы**: $0.00 (бесплатно)
- **ОБЩАЯ СТОИМОСТЬ**: **~$1.64**

## Дополнительные соображения

### Масштабирование:
- При аудите нескольких услуг: каждая дополнительная услуга добавляет ~$0.975
- При аудите 10 услуг: $1.64 + (10 × $0.975) = **~$11.39**

### Оптимизация:
- llms.txt анализируется только один раз для всего сайта (не для каждой страницы)
- PageSpeed API имеет лимит 25,000 запросов/день, что достаточно для ~12,500 страниц в день

### Рекомендуемая наценка для клиента:
- **Полный аудит клиники**: $1.64 → продавать за **$15-25** (наценка 900-1500%)
- **Аудит услуги**: $0.975 → продавать за **$10-15** (наценка 1000-1500%)
- **Пакет (клиника + услуга)**: $2.615 → продавать за **$20-35** (наценка 700-1300%)

## Примечания

1. **Цены на OpenAI могут меняться** - рекомендуется проверять актуальные цены на [openai.com/pricing](https://openai.com/pricing)

2. **PageSpeed API бесплатен**, но имеет лимиты:
   - 25,000 запросов в день на один API ключ
   - При превышении лимита запросы будут отклоняться

3. **Дополнительные расходы** (не включены в расчет):
   - Хостинг сервера
   - Хранение данных в базе данных
   - Трафик (обычно минимальный)

4. **Время выполнения**:
   - Полный аудит клиники: ~20-30 секунд
   - Аудит услуги: ~15-25 секунд
   - Основное время уходит на PageSpeed API (может занимать 10-20 секунд на запрос)
