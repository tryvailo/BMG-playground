# Сравнение TSV и Excel форматов для базы данных городов

## Анализ форматов

### TSV формат (`medical_centers_ukraine.tsv`)

**Преимущества:**
- ✅ **Простота парсинга** - текстовый формат, не требует внешних библиотек (xlsx)
- ✅ **Меньший размер файла** - только текст, без метаданных Excel
- ✅ **Чистые данные** - колонка "Місто / Регіон" содержит только названия городов (без "/ Регіон")
- ✅ **Легко редактировать** - можно открыть в любом текстовом редакторе
- ✅ **Быстрее обрабатывается** - простой split по табуляции
- ✅ **41 уникальный город** из 378 записей

**Недостатки:**
- ⚠️ Меньше записей (378 vs 200+ в Excel)
- ⚠️ Может быть менее структурированным для сложных данных

### Excel формат (`medical_centers_ukraine_200plus.xlsx`)

**Преимущества:**
- ✅ Больше записей (200+)
- ✅ Более структурированный формат
- ✅ Поддержка форматирования

**Недостатки:**
- ❌ Требует библиотеку `xlsx` (дополнительная зависимость)
- ❌ Больший размер файла
- ❌ Более сложный парсинг (нужно обрабатывать разные форматы ячеек)
- ❌ Колонка может содержать "Город / Регион" (нужно извлекать только город)

## Рекомендация

**TSV формат лучше подходит для онбординга**, потому что:

1. **Простота** - не нужны дополнительные зависимости для парсинга
2. **Чистота данных** - города уже в правильном формате
3. **Скорость** - быстрее обрабатывается
4. **41 город** - достаточно для выбора в онбординге

## Использование

### Импорт из TSV (рекомендуется):
```bash
pnpm import-ukraine-cities-tsv
```

### Импорт из Excel:
```bash
pnpm import-ukraine-cities
```

## Структура TSV файла

```
Назва клініки | Сайт | Місто / Регіон | Кількість послуг
On Clinic     | ...  | Біла Церква    | 47
Dr.David      | ...  | Біла Церква    | 16
...
```

**Колонка "Місто / Регіон"** содержит только названия городов:
- Біла Церква
- Бориспіль
- Бровари
- Київ
- Львів
- и т.д.

## Вывод

TSV формат идеально подходит для локальной базы данных городов в онбординге:
- Простой и быстрый парсинг
- Чистые данные без дополнительной обработки
- Не требует внешних библиотек
- Легко поддерживать и обновлять
