# Полный расчет стоимости аудита и анализа

## Обзор

Этот документ содержит полный расчет стоимости всех компонентов системы аудита:
1. **Технический аудит** (PageSpeed, HTML анализ, Schema, LLMS.txt)
2. **Поиск дубликатов контента** (Firecrawl + алгоритм анализа)

---

## Часть 1: Технический аудит

### Используемые API и сервисы

#### 1. Google PageSpeed Insights API
- **Статус**: Бесплатный (с лимитами)
- **Лимиты**: 
  - 25,000 запросов в день (на один API ключ)
  - Нет платы за использование в пределах лимита
- **Использование в аудите**: 
  - 2 запроса на каждый URL (desktop + mobile)
  - Для клиники: 2 запроса (главная страница)
  - Для услуги: 2 запроса (страница услуги)

#### 2. OpenAI API (GPT-4o-mini)
- **Модель**: gpt-4o-mini
- **Цены** (на декабрь 2025):
  - Input: $0.150 за 1M токенов (~$0.00015 за 1K токенов)
  - Output: $0.600 за 1M токенов (~$0.0006 за 1K токенов)
  - *Примечание: OpenAI сохранил стабильные цены на GPT-4o-mini в 2025 году*
- **Использование в аудите**:
  - Анализ llms.txt: ~1,500-2,500 токенов (input) + ~500-800 токенов (output)
  - Анализ технического аудита: ~2,000-3,500 токенов (input) + ~800-1,200 токенов (output)

#### 3. HTTP запросы (бесплатно)
- robots.txt, sitemap.xml, llms.txt, HTML страницы
- Обычные HTTP запросы без платы

### Расчет стоимости технического аудита

#### Полный аудит клиники (главная страница)

**Компоненты:**
1. PageSpeed API (desktop + mobile): **$0** (бесплатно)
2. HTML парсинг, файлы: **$0** (бесплатно)
3. Анализ llms.txt (если есть):
   - Input: ~2,000 токенов × $0.00015 = **$0.0003**
   - Output: ~700 токенов × $0.0006 = **$0.0004**
   - Итого: **~$0.0007**
4. Анализ технического аудита:

   - Итого: **~$0.00105**

**Общая стоимость полного аудита клиники: ~$0.00175**

#### Аудит одной услуги (страница услуги)

**Компоненты:**
1. PageSpeed API (desktop + mobile): **$0** (бесплатно)
2. HTML парсинг, файлы: **$0** (бесплатно)
3. Анализ llms.txt: **$0** (не используется для услуги)
4. Анализ технического аудита:
   - Input: ~2,500 токенов × $0.00015 = **$0.000375**
   - Output: ~900 токенов × $0.0006 = **$0.00054**
   - Итого: **~$0.000915**

**Общая стоимость аудита услуги: ~$0.000915**

#### Комбинированный пакет (клиника + 1 услуга)

**Общая стоимость: ~$0.002665** (~$0.003)

### Детальный расчет токенов

#### Анализ llms.txt

**Input токены:**
- System prompt: ~200 токенов
- User prompt template: ~200 токенов
- llms.txt content: ~1,000-2,000 токенов (зависит от размера файла)
- **Итого Input**: ~1,400-2,400 токенов (среднее: ~2,000)

**Output токены:**
- JSON структура: ~500-800 токенов
- Среднее: ~700 токенов

**Стоимость:**
- Input: 2,000 × $0.00015 = $0.0003
- Output: 700 × $0.0006 = $0.00042
- **Итого: ~$0.00072**

#### Анализ технического аудита

**Input токены:**
- System prompt: ~300 токенов
- User prompt template: ~200 токенов
- Форматированные данные аудита: ~1,500-3,000 токенов
- **Итого Input**: ~2,000-3,500 токенов (среднее: ~3,000)

**Output токены:**
- JSON структура с анализом: ~800-1,200 токенов
- Среднее: ~1,000 токенов

**Стоимость:**
- Input: 3,000 × $0.00015 = $0.00045
- Output: 1,000 × $0.0006 = $0.0006
- **Итого: ~$0.00105**

### Масштабирование

#### Для N услуг:

**Формула:**
```
Стоимость = $0.00175 (клиника) + N × $0.000915 (услуга)
```

**Примеры:**
- 5 услуг: $0.00175 + 5 × $0.000915 = **~$0.0063**
- 10 услуг: $0.00175 + 10 × $0.000915 = **~$0.0109**

## Часть 2: Поиск дубликатов контента

### Используемые сервисы

1. **Firecrawl API** - для сканирования страниц сайта
2. **Локальный алгоритм** - для анализа дубликатов (бесплатно)

### Firecrawl API Pricing

#### Тарифные планы (примерные, на декабрь 2025)

**⚠️ ВАЖНО**: Точные цены нужно проверять на https://firecrawl.dev/pricing

#### Starter Plan (примерно)
- **Цена**: $20-30/месяц
- **Лимит**: ~1,000 страниц/месяц
- **Стоимость за страницу**: ~$0.02-0.03

#### Pro Plan (примерно)
- **Цена**: $99-149/месяц
- **Лимит**: ~10,000 страниц/месяц
- **Стоимость за страницу**: ~$0.01-0.015

#### Business Plan (примерно)
- **Цена**: $299+/месяц
- **Лимит**: ~50,000+ страниц/месяц
- **Стоимость за страницу**: ~$0.006-0.01

#### Pay-as-you-go
- Обычно: $0.01-0.03 за страницу
- Зависит от объема и договоренностей

#### Сценарий 1: Starter Plan (если есть подписка)
- **50 страниц** = входит в месячный лимит (~1,000 страниц)
- **Стоимость одного сканирования**: $0 (в рамках плана)
- **Маржинальная стоимость**: $0.02 × 50 = **$1.00**

#### Сценарий 2: Pay-as-you-go (без подписки)
- **50 страниц** × $0.02 = **$1.00**
- **50 страниц** × $0.03 = **$1.50** (максимум)

#### Сценарий 3: Pro Plan (если есть подписка)
- **50 страниц** = входит в месячный лимит (~10,000 страниц)
- **Стоимость одного сканирования**: $0 (в рамках плана)
- **Маржинальная стоимость**: $0.01 × 50 = **$0.50**

---

### Итоговая стоимость поиска дубликатов

#### Для одного сканирования (50 страниц):
- **Минимум**: $0.50 (Pro Plan, маржинальная стоимость)
- **Средняя**: $1.00-1.50 (Starter Plan или Pay-as-you-go)
